# Gesture-Based Air Canvas

A real-time virtual drawing application built using Python, OpenCV, and MediaPipe. Users can draw on a virtual canvas using their index finger tracked through a webcam.

## Features

* Real-time hand tracking
* Draw using finger gestures
* Multiple color options:

  * Blue
  * Green
  * Red
  * Yellow
* Eraser tool
* Clear canvas functionality
* Adjustable brush size 
* Save drawings as PNG images 
* Live webcam interface

## Technologies Used

* Python
* OpenCV
* MediaPipe
* NumPy

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Gesture-Based-Air-Canvas.git
cd Gesture-Based-Air-Canvas
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```

## Controls

| Action              | Control             |
| ------------------- | ------------------- |
| Draw                | Move index finger   |
| Select Color        | Touch color button  |
| Eraser              | Touch eraser button |
| Clear Canvas        | Touch clear button  |
| Increase Brush Size | +                   |
| Decrease Brush Size | -                   |
| Save Drawing        | S                   |
| Exit                | ESC                 |
